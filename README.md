# TradeBoard

A dark-themed trading dashboard built with Next.js, TypeScript, Tailwind CSS, and TanStack Query. It displays live cryptocurrency market data from the CoinGecko API.

### !!!!!! if the project doesnt work for whatever resone you can use this link
### https://trade-board-orcin.vercel.app/dashboard


## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- TanStack Query
- CoinGecko API

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/tradeboard.git
```

### 2. Navigate to the project

```bash
cd tradeboard
```

### 3. Install dependencies

```bash
npm install
```





### 4. Run the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:3000
```
